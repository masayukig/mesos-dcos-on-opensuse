=========================================
Mesos, DC/OS on openSUSE つかってみた
=========================================

Built with beamer latex

Install texlive with most of the plugins via your distro

to build just run::

  % pdflatex mesos-dcos-on-opensuse.tex

the output will be mesos-dcos-on-opensuse.pdf
